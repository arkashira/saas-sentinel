# breakeven.md

## Unit Economics & Break-even Analysis

### Cost per Active User
- **Compute**: $0.10/user/month
- **Storage**: $0.05/user/month
- **Bandwidth**: $0.02/user/month
- **Total Cost per Active User**: **$0.17/user/month**

### Pricing Tiers
| Tier             | Price ($/mo) | Features                                                                                     |
|------------------|--------------|----------------------------------------------------------------------------------------------|
| **Basic**        | $15          | User insights dashboard, Basic analytics, Email support                                      |
| **Pro**          | $30          | All Basic features, Advanced analytics, API access, Priority email support                  |
| **Enterprise**    | $60          | All Pro features, Custom integrations, Dedicated account manager, SLA support               |

### Customer Acquisition Cost (CAC) Range
- Estimated CAC: **$100 - $150** per user

### Lifetime Value (LTV) Estimate
- Average Monthly Revenue per User (ARPU) for Pro Tier: $30
- Average Customer Lifespan: 24 months
- **LTV = ARPU × Customer Lifespan = $30 × 24 = $720**

### Break-even Users Count
- Fixed Costs (e.g., salaries, marketing, infrastructure): $10,000/month
- Contribution Margin per User (Price - Cost): 
  - Basic: $15 - $0.17 = $14.83
  - Pro: $30 - $0.17 = $29.83
  - Enterprise: $60 - $0.17 = $59.83
- Break-even Users Count:
  - Basic: $10,000 / $14.83 ≈ **674 users**
  - Pro: $10,000 / $29.83 ≈ **335 users**
  - Enterprise: $10,000 / $59.83 ≈ **167 users**

### Path to $10K MRR
To achieve $10K MRR, the following combinations can be considered:

#### Scenario 1: Basic Tier
- Price: $15/user
- Required Users: $10,000 / $15 ≈ **667 users**

#### Scenario 2: Pro Tier
- Price: $30/user
- Required Users: $10,000 / $30 ≈ **334 users**

#### Scenario 3: Enterprise Tier
- Price: $60/user
- Required Users: $10,000 / $60 ≈ **167 users**

### Summary
To reach $10K MRR, the most feasible path with the least user count is targeting the **Enterprise Tier** with **167 users**. However, a balanced approach across tiers can also be pursued to diversify revenue streams and mitigate risks.
